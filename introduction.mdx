## What's Included

This repository contains configuration files for:

- **Shell Environment**: Zsh configuration with Oh My Zsh framework, Powerlevel10k theme, and productivity-focused plugins
- **Terminal Multiplexer**: Tmux setup with custom keybindings, theme, and TPM plugin management
- **File Manager**: Yazi configuration with rose-pine theme, advanced keymaps, and optimized settings
- **CLI Tools**: Integration with modern command-line tools including eza (better ls), zoxide (better cd), fzf (fuzzy finder), fd (better find), and bat (better cat)
- **Development Tools**: Custom OpenCode agent for debugging assistance and enhanced development workflows

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Magiclovekorean/dotfiles.git ~/dotfiles
    ```
  </Step>
  <Step title="Install dependencies">
    Install the required tools and frameworks. See the [Installation](/installation) guide for detailed instructions.
  </Step>
  <Step title="Create symlinks">
    Link the configuration files to your home directory:
    ```bash
    ln -s ~/dotfiles/.zshrc ~/.zshrc
    ln -s ~/dotfiles/.tmux.conf ~/.tmux.conf
    ln -s ~/dotfiles/.config ~/.config
    ```
  </Step>
  <Step title="Start using">
    Restart your shell or source the configuration:
    ```bash
    source ~/.zshrc
    ```
  </Step>
</Steps>
